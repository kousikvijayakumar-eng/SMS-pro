# 🏫 SMS Pro — Advanced Student Management System

A full-stack web application built as a final year BCA project.
Manage students, attendance, marks, fees, homework, and notices
with role-based access for Admin, Teacher, and Student.

## ✨ Features
- 📧 Email OTP login (no passwords)
- 👥 3 roles — Admin / Teacher / Student
- 📋 Student registration & profiles
- 📅 Attendance tracking with % summary
- 📊 Subject-wise marks & auto grade (A+/A/B/C/D)
- 💰 Fee management (Paid / Pending / Partial)
- 📢 Notice board with priority levels
- 📚 Homework assignment & submission

## 🛠 Tech Stack
| Layer | Technology |
|-------|-----------|
| Frontend | HTML, CSS, Vanilla JS |
| Backend | Node.js + Express.js |
| Database | Supabase (PostgreSQL) |
| Auth | Email OTP + JWT |
| Hosting | Render (backend) + Vercel (frontend) |
